# 挑战 1：两数之和

## 问题描述

编写一个函数 `Sum`，该函数接受两个整数并返回它们的和。

## 函数签名

```go
func Sum(a int, b int) int
```

## 输入格式

- 两个整数 `a` 和 `b`。

## 输出格式

- 一个整数，表示 `a` 和 `b` 的和。

## 约束条件

- `-10^9 <= a, b <= 10^9`

## 示例输入和输出

### 示例输入 1

```
2, 3
```

### 示例输出 1

```
5
```

### 示例输入 2

```
-5, 10
```

### 示例输出 2

```
5
```
